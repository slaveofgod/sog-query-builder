Init State.
Name Type Description
state Object Query builder state.

Example
var State = new sogqb.State([{
    {
        type: 'expression',
        field: 'user.first_name',
        label: 'Alexey',
        value: 1,
        operator: 'equal'
    }, {
        type: 'conjunction',
        operator: 'and'
    }, {
        type: 'expression',
        field: 'user.last_name',
        label: 'Bob',
        value: 1,

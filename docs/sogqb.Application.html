Class: Application

sogqb.Application

Query Builder Application.

new sogqb.Application (container, entity, scheme, state, theme)

Init Application.
Name Type Default Description
container String The container of query string.
entity String The current entity.
scheme Object null The scheme of query.
state Object null Query builder state. If this value is null, it will be used data from url.
theme String material Theme.
Example
var queryBuilder = new sogqb.Application('user-query-builder', 'user');
